# **App Name**: DevFolio

## Core Features:

- Hero Section: Displays a professional photo, name, role, and a personal motto.
- About Me: Showcases a brief professional bio, technical skills, and soft skills.
- Projects Portfolio: Displays projects with descriptions, screenshots, technologies, and links to repositories and live demos.
- Experience & Education: A timeline of previous jobs, internships, collaborations, and academic background, including relevant certifications.
- Contact Form: A form for users to send messages, plus links to GitHub, LinkedIn, and email.
- Downloadable CV: Provides a downloadable CV in PDF format.

## Style Guidelines:

- Primary color: Dark blue (#30475E) to convey professionalism and trust.
- Background color: Light gray (#F0F0F0) for a clean, minimal look.
- Accent color: Teal (#548CA8) to highlight key elements and calls to action.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and readable experience.
- Use flat, minimalist icons for technologies (HTML, CSS, JavaScript, React, Python) and social media links.
- Divide the React components into Header, Body, and Footer for simplicity and easier code maintenance as requested.
- Use subtle hover and scroll animations to add a modern touch.
